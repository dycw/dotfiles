# `dotfiles`

Dotfiles

## Install

### GitHub

```sh
curl -fsSL \
  https://raw.githubusercontent.com/dycw/dotfiles/refs/heads/master/setup.sh \
  | sh
```

### Gitea

```sh
curl -fsSL \
  https://gitea.ai/derek/dotfiles/raw/branch/master/setup.sh \
  | DOTFILES_REPO=https://gitea.ai/derek/dotfiles.git sh
```
